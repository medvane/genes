= Real-Time Review on Genes

== Set Up

 $ git clone git://github.com/seouri/rtreview.git
 $ cd rtreview
 $ bundle install --without production
 $ cp config/database.yml.mysql config/database.yml
 $ rake rtreview:setup
 
== Update Base Data

 $ rake rtreview:update:all

== Run the Background Worker

 $ rake jobs:work
